# Deploy Setup Skill

A skill for automating deployment to Vercel/Netlify.

---

## Features

- Vercel/Netlify project creation guide
- Environment variable configuration guide
- Automatic deployment via GitHub Actions
- Preview deployments (Pull Requests)

---

## Execution Flow

1. Check project structure
2. Choose Vercel or Netlify
3. Generate configuration files (vercel.json / netlify.toml)
4. Guide environment variable configuration
5. Verify deployment
